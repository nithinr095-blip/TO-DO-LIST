

 📝 TO DO LIST

A simple Django-based To Do List application that allows users to add, update, and delete tasks easily.

---

🚀 Features

* ➕ Add tasks
* ✏️ Update tasks
* ❌ Delete tasks
* 🎨 Simple and clean UI using CSS
* 📋 Display all tasks dynamically

---

 🛠️ Built With

* Python
* Django
* HTML
* CSS

 ⚙️ Installation

 1️⃣ Clone the repository

```bash
git clone https://github.com/nithinr095-blip/TO-DO-LIST.git
```

 2️⃣ Go to project folder

```bash
cd TO-DO-LIST
```

 3️⃣ Create virtual environment

```bash
python -m venv env
```

 4️⃣ Activate virtual environment

 Windows

```bash
env\Scripts\activate
```

 Mac/Linux

```bash
source env/bin/activate
```

 5️⃣ Install Django

```bash
pip install django
```

 6️⃣ Run migrations

```bash
python manage.py migrate
```

 7️⃣ Start server

```bash
python manage.py runserver
```

---

 🌐 Open in Browser

```bash
http://127.0.0.1:8000/list/
```
